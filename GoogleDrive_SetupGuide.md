# Google Drive 同期機能のセットアップ手順 ☁️🎵

このアプリは、Google Drive内の音楽ファイル（mp3など）を直接ストリーミング再生する機能を持っています。
セキュリティ上、あなた個人のGoogleアカウントに紐付いた「APIキー」と「クライアントID」を設定する必要があります。

以下の手順に沿ってキーを取得し、プログラムを書き換えてください！

---

## ステップ1：Google Cloud プロジェクトの作成

1. [Google Cloud Console](https://console.cloud.google.com/) にアクセスし、Googleアカウントでログインします。
2. 画面上部の「プロジェクトを選択」を押し、「新しいプロジェクト」をクリックします。
3. プロジェクト名（例: `Glass Music Sync`）を入力して「作成」を押します。

## ステップ2：Google Drive API の有効化

1. 作成したプロジェクトの画面で、左メニューから「**APIとサービス**」＞「**有効なAPIとサービス**」を開きます。
2. 画面上部の「**+ API とサービスの有効化**」をクリックします。
3. 検索窓で「**Google Drive API**」と検索し、選択して「**有効にする**」を押します。

## ステップ3：OAuth同意画面の設定

1. 左メニューから「**APIとサービス**」＞「**OAuth 同意画面**」を開きます。
2. User Type で「**外部 (External)**」を選び「作成」を押します。
3. 以下の必須項目だけ入力して、一番下までスクロールし「保存して次へ」を押します。
   - **アプリ名**: `Glass Music Player` など
   - **ユーザーサポートメール**: （あなたのアドレスを選択）
   - **デベロッパーの連絡先情報**: （あなたのアドレスを入力）
4. 以降の画面（スコープ、テストユーザー）は何も入力せず「保存して次へ」を進め、「ダッシュボードに戻る」を押します。
5. 「**アプリを公開**」ボタンを押して、テスト状態から本番へ移行（In production）させます。

## ステップ4：認証情報（キー）の作成と取得

1. 左メニューから「**認証情報**」を開きます。
2. **【APIキーの取得】**
   - 画面上部の「**+ 認証情報を作成**」＞「**APIキー**」をクリックします。
   - 作成されたAPIキー（アルファベットと記号の長い文字列）をコピーして、どこかにメモしておきます。
3. **【クライアントIDの取得】**
   - もう一度「**+ 認証情報を作成**」＞「**OAuth クライアント ID**」をクリックします。
   - アプリケーションの種類: 「**ウェブ アプリケーション**」を選択します。
   - 名前: 初期値のままでOK
   - **承認済みの JavaScript オリジン（重要）**: 「+ URIを追加」を押し、アプリを動かすURLを登録します。
     - PCローカル用: `http://localhost` および `http://127.0.0.1`
     - スマホ用（GitHub Pages）: `https://[あなたのユーザー名].github.io`
   - 「作成」ボタンを押します。
   - 表示された「**クライアント ID**」をコピーしてメモします。

---

## ステップ5：ソースコードの書き換え

準備が整いました！最後にアプリのコードを書き換えます。

1. このプロジェクトフォルダに入っている **`index.html`** をテキストエディタ（VSCodeやメモ帳）で開きます。
2. ファイルの **一番下の方（1420行目付近）** にある以下の部分を見つけてください。

```javascript
        // ！！！事前にGoogle Cloud Consoleから取得した値に必ず書き換えてください！！！
        const CLIENT_ID = 'YOUR_CLIENT_ID';
        const API_KEY = 'YOUR_API_KEY';
```

3. `'YOUR_CLIENT_ID'` と `'YOUR_API_KEY'` の文字を、さきほど取得したあなたの文字列に置き換えます。
   ※ 両端のクォーテーション（`'`）は消さないように注意してください！

4. ファイルを保存（`Ctrl + S`）します。

---

## 完了 🎉

これですべての準備が完了しました！
1. ブラウザで `index.html` を開くか、GitHubへ再度Pushします。
2. 画面の右下（音量の横）にある **雲のボタン（Driveに接続）** を押します。
3. Googleログイン画面が出るので、自分のアカウントでログインして「許可」を押します。
4. 自動的にあなたのGoogle Drive内に `music` というフォルダが作られます！そこにMP3ファイルを入れておくと、このアプリからどこでも聴けるようになります。
