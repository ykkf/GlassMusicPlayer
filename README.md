# Glass Music Player 💿

ガラス風デザインの音楽プレイヤーです。PC（Windows専用アプリ）とスマートフォン（PWA）の両方に対応しています。

## 🌟 特徴
- **Glassmorphism UI**: 美しい磨りガラスのような透過デザイン
- **Waveform Visualization**: 再生中の音に合わせた波形アニメーション
- **Always On Top**: 作業中も常に最前面に表示可能（ピン留め機能）
- **Mini Mode**: 邪魔にならないコンパクトなモードへ切替可能
- **PWA Ready**: スマホのブラウザから「ホーム画面に追加」してアプリとして利用可能

## 💻 PC版 (Windows) の使い方
1. [Releases](https://github.com/YOUR_USERNAME/YOUR_REPO_NAME/releases) から最新の `Glass Music Player-win32-x64.zip` をダウンロードします。
2. 展開したフォルダ内の `Glass Music Player.exe` を実行してください。

## 📱 スマホ版 (PWA) の使い方
1. GitHub Pages でホストされたURL（`https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/`）にスマホのブラウザでアクセスします。
2. ブラウザのメニューから「ホーム画面に追加」を選択してください。
